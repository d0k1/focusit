Octopress Cheat Sheet

Dec 4th, 2016 10:08 pm

Octopress posting workflow:

  1. rake new_post["Post title"]. Note, if you are zsh user don’t forget to escape [ and ]
  2. rake generate
  3. git add .
  4. git commit -am 'Post title'
  5. git push origin source
  6. rake deploy
